Detailed Description

A helper class to parameters from "options" configuration section.

var config = ConfigParams.FromTuples( ...
"options.param1", "ABC",
"options.param2", 123 );
var options = OptionsResolver.Resolve(config, false); // Result: param1=ABC;param2=123
